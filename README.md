Commission Management System

A modern commission and sales tracking system built with Flask and SQLite.

Features

- Customer management
- Supplier management
- Product inventory
- Purchase tracking
- Order management
- Payment tracking
- Commission calculation
- Activity logs
- Reports dashboard
- User authentication

Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

Installation

git clone https://github.com/USERNAME/commission-system.git

cd commission-system

pip install -r requirements.txt

python app.py
